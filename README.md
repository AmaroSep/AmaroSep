### Featured project

#### [SparkFRET](https://github.com/amarosep/SparkFRET)
> End-to-end pipeline for automated FRET sparkle detection and quantification in widefield microscopy.

- Custom **Cellpose** model trained on FRET biosensor images
- **Streamlit** hub: mask generation → cell segmentation → FRET quantification → statistics
- Replaces manual CellProfiler workflows with a single-click Python pipeline
- Designed for Cytation plate-reader images (96-well format)
